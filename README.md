## 🎯 Sobre este Estudo
Este repositório foi criado para fins estritamente didáticos, focando na prática de conceitos fundamentais de **HTML5** e **CSS3**.

### Requisitos Implementados:
* **Foto de Perfil Arredondada**: Prática de `border-radius: 50%` e `object-fit: cover`.
* **Incorporação de Vídeo**: Uso da tag `<iframe>` com o formato `embed` do YouTube para garantir a exibição interna.
* **Alinhamento Horizontal**: Centralização de cabeçalhos e elementos de mídia usando CSS.
* **Estilização Dark**: Aplicação de uma paleta de cores personalizada (preto e vermelho).

## 💻 Como visualizar localmente
O projeto foi desenvolvido utilizando o VS Code. Para rodar:
1. Clone o repositório.
2. Abra o arquivo `index.html`.
3. Utilize a extensão **Live Server** (acessível localmente em `http://127.0.0.1:5500/`).
