# Projeto de Currículo Web - Prática de Integração de Mídia

Este projeto foi desenvolvido para fins estritamente acadêmicos e de estudo, com o objetivo de praticar conceitos fundamentais de Front-End.

## 🚀 Objetivos do Estudo
O foco principal desta atividade foi aprender e aplicar:

* **Manipulação de Imagens com CSS**: Uso da propriedade `border-radius: 50%` para criar fotos de perfil em formato arredondado.
* **Incorporação de Vídeos**: Uso da tag `<iframe>` para integrar conteúdo do YouTube diretamente na página, garantindo o funcionamento através do formato `embed`.
* **Alinhamento e Layout**: Aplicação de `text-align: center` e controle de margens para garantir que títulos, subtítulos e mídias estejam perfeitamente alinhados horizontalmente.
* **Estilização Temática**: Criação de uma interface com modo escuro (*dark mode*) personalizada com referências visuais de cultura pop (Marceline/Anime).

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3
